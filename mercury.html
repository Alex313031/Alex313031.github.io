Mercury -
The fastest firefox fork on Earth.

Firefox fork for Linux and Windows named after Firefox's version control system, Mercurial, and the liquid element No. 80.

Highlights  

  • Compiler optimizations include SSE4.2, AVX, AES, and modifications to CFLAGS, LDFLAGS, thinLTO flags, and PGO, as well as other compiler flags.
  • Learn more about these compiler optimizations and how they work Here.
  • UI Changes and Patches for Linux and Windows that fix bugs, enhance useability, and strengthen privacy/security.

  This nets a 8-40% performance improvement over vanilla Chromium, depending on the benchmark and OS. See PATCHES.md for a detailed overview of all modifications compared to vanilla Chromium regarding performance, security and useability.

Installation

The releases come in a tarball. For Windows, just unpack anywhere, and make a shortcut to mercury.exe if you wish. For Linux, it can be used on any distribution. To add to your system:

tar -xvf mercury-versionnumber.tar.gz &&
run ./mercury-bin or ./RUN.sh
----------------------------------------
You can create a .desktop file using
the sample one provided in the tarball.
