## Thorium Dino Game

The  T-Rex Runner Game extracted from the Chromium Offline Error Page. (Can be run in Thorium/Chromium by going to chrome://dino/ )

See the [source](https://cs.chromium.org/chromium/src/components/neterror/resources/) from Chromium.

[Try it!](https://thorium.rocks/game/game.html) :smile: at thorium.rocks.

![Thorium Dino](../imgs/Dino.png)

In Thorium, it is directly from source. \
For the web version here and on [thorium.rocks](https://thorium.rocks/), it was created based on https://github.com/wayou/t-rex-runner
