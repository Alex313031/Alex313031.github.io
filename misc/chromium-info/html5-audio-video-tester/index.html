 HTML5 Audio/Video Tester

A free online file type player and MIME type tester, mainly for Chromium and Firefox browsers.

How to use this tool?

This is a simple page to test only HTML <audio> and <video> tags without Javascript.

By default here, the MIME type is undefined but you can use a site query string to force it.

Examples to test a media file + MIME type:
  • m=video or m=audio (if you do not know MIME type → It will only define HTML tag)
  • m=video/ogg; codecs="theora, vorbis" (if you want to test a MIME type → It will define both MIME type and HTML tag)
  • l=1 (to enable audio/video loop)

Example to test only a MIME type, without a media file (JS needed):
  • m=video/ogg; codecs="theora, vorbis"
