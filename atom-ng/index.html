Installing

Simply go to the Releases page. The .deb files can be used on Debian/Ubuntu & derivatives. The .tar.gz files can be used for any linux distribution; use the RUN.sh file within to launch it.
For Windows, download the .zip and run the atom-ng shortcut inside.
