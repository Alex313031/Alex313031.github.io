Thorium -
The fastest browser on Earth.

Chromium fork for Linux, MacOS, Raspberry Pi, and Windows named after radioactive element No. 90.

Highlights

  • Compiler modifications include SSE4.2, AVX, AES, and Cflags, thinLTO flags, and import_instr_limit flags set to /O3, 3, and 30 respectively.
  • LLVM LOOP optimizations as -mllvm flags

 This nets a 2-8% performance improvement, depending on the benchmark and OS. See PATCHES.md for a detailed overview of all modifications compared to vanilla Chromium regarding performance, security and useability.
 See also: ThoriumOS

Installation/Auto-Update

The repo can be used on any distribution that supports .deb packaging.
It is located at https://dl.thorium.rocks/. To add to your system:

wget https://dl.thorium.rocks/debian/dists/stable/thorium.list
sudo mv thorium.list /etc/apt/sources.list.d/
sudo apt update
sudo apt install thorium-browser-unstable
