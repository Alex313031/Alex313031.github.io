To link a sound file to a document

  1. Open the sound file you want to link.
  2. On the Edit menu, click Copy.
  3. Open the document in which you want the sound, and then click where you want it to play.
  4. On the Edit menu, click Paste Special.
  5. Click the format you want to use, and then click Paste Link.

Note

  • If the Edit menu does not contain Paste Special, your program does not support linking.
