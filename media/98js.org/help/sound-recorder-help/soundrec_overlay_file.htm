To overlay (mix) sound files

  1. Move the slider to the place in the file where you want to overlay the sound file.
  2. On the Edit menu, click Mix with File.
  3. Enter the name of the file you want to mix.
Note
  • You can overlay an uncompressed sound file only. If you do not see the green line in Sound Recorder, the file is compressed and you cannot modify it unless you change the sound quality.
