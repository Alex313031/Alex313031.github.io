ThoriumOS -
ChromiumOS for the masses.

ChromiumOS fork with Thorium Browser, x264/x265 codecs, Widevine, Kernel 5.15, Linux Firmware/Modules support, Nouveau, Intel/AMD microcode, and extra packages.

Installation

The repo hosts .7z files in the Releases section with a .bin file inside.
It is located at: https://github.com/Alex313031/ThoriumOS/

To install:
- Flash the .bin file to a USB Flash Drive (8GB or larger). You can use DD, Etcher, the ChromeBook Recovery Utility, Rufus, or cros_flash.
- Install to your device by following CrOS Installation. However when running /usr/sbin/chromeos-install append --dst /dev/sdxx, where sdxx is your hard drive i.e. sda1.
