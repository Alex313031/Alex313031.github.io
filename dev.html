Thorium Developer Tools

   Developer Tools for Thorium Builders

| For Thorium Development and the Chromium Code Search:

  • Thorium Documentation Page  
  • Thorium NetLog Viewer  
  • Skiafier  
  • Vector Icon Viewer  
  • Chromium Vector Icons - Extension which draws vector .icon files.  
  • Thorium Code Search Themer - Themer for Chromium Code Search.  
  • Chromium Dev Editor - Code/Text Editor App for Thorium.  
  • Release Notes  

| For Web Development, and Testing Thorium or GitHub Download Stats:

  • Speedometer 2.1 (Custom Fork) - Benchmark for your browser.  
  • Octane 2.1 (Custom Fork) - Javascript benchmark for browser.  
  • JetStream 2 (Custom Fork) - JavaScript and WebAssembly benchmark suite.  
  • GitHubStat v2 - Get information about download counts for GitHub releases, release author and date of publishing.  
  • Resources Saver - Extension for one click downloading of all webpage resources files while keeping folder structures.  
  • HTML5 Audio/Video Tester   - HTML5 audio and video tester to test file type and MIME type.  
  • WebGL Tester    - Also see Here and Here to verify WebGL 1 and 2, respectively.
  • WebGPU Tester    - Test page for WebGPU. It can also test WebGL 1 & 2.
  • JPEG XL Tester    - Test Page for JPEG XL. Thorium has support for this by default, unlike any other Chromium-based browser!
  • HEVC/H.265/AC3/E-AC3 Video Tester    - Test Page for H.265 video (aka HEVC), AC3 audio, and E-AC3 audio. Thorium has support for these by default, unlike any other browser!

- More coming soon!
